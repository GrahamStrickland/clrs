In order to account for an empty subarray, we need only specify that
\textit{left-sum},
\textit{right-sum}, or \textit{cross-sum} be greater than 0 in the conditional
\textbf{if} \ldots \textbf{else} statement of our algorithms. We may
then add a final
\textbf{else} clause allowing us to return an empty subarray if none
of the other
conditionals return a true value.
