We are looking for the interval where
\begin{equation*}
    \begin{aligned}
        8n^2 &< 64n\, \text{lg}\, n\\
        \Rightarrow n &< 8\, \text{lg}\, n\\
        \Rightarrow \frac{n}{\text{lg}\, n} &< 8\\
        \Rightarrow -n \, \text{lg}\, n &< 8\\
        \Rightarrow n \, \text{lg}\, n &\geq 8.
    \end{aligned}
\end{equation*}
For $n < 44$, we have insertion sort faster than merge sort.
